# Guia Rápido — Dashboard de Vendas (Python + Streamlit)

Este guia explica, de forma curta, como correr o projeto em Windows, macOS ou Linux, **sem** depender do Visual Studio Code.

---

## 1) Pré-requisitos
- **Python 3.11+** instalado  
  - Verifica com: `python --version` (Windows) ou `python3 --version` (macOS/Linux)
- Acesso à internet para instalar dependências

---

## 2) Obter o projeto
Clona o repositório ou faz download do ZIP:
```bash
git clone https://github.com/<teu-utilizador>/dashboard_dados.git
cd dashboard_dados
